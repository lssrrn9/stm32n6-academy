1 Introducción a la NPU Ethos-U55

¿Qué es una NPU?

Una Unidad de Procesamiento Neural (NPU) es un procesador especializado diseñado específicamente para acelerar las operaciones de redes neuronales artificiales. A diferencia de una CPU de propósito general que ejecuta instrucciones secuenciales, una NPU está optimizada para el paralelismo masivo requerido por la inferencia de modelos de deep learning.

Ethos-U55: microNPU de ARM

La Ethos-U55 es una microNPU diseñada por ARM específicamente para microcontroladores. Integrada en la familia STM32N6, ofrece:

  • Hasta 512 GOPS de rendimiento de inferencia
  • Soporte nativo para cuantización INT8
  • Interfaz directa con Cortex-M55 a través del bus AXI
  • Memoria SRAM compartida para tensores intermedios
  • Consumo ultra-bajo: ideal para aplicaciones de batería

Ventajas de la inferencia en el Edge

  • Latencia mínima: respuesta en menos de 1ms, sin dependencia de red
  • Privacidad de datos: los datos sensibles nunca salen del dispositivo
  • Sin dependencia de cloud: operación autónoma offline
  • Consumo optimizado: la NPU consume menos energía que la CPU para inferencia
  • Seguridad: menor superficie de ataque al no requerir conectividad

2 Flujo de Trabajo STM32Cube.AI

STM32Cube.AI es el toolkit oficial de ST para convertir modelos de redes neuronales pre-entrenados en código C optimizado que puede ejecutarse en microcontrolores STM32. El flujo completo consta de 8 pasos:

1
Entrenar modelo en Python
Usar TensorFlow/Keras para entrenar el modelo con tus datos. Emplear un modelo "tiny" (pocas capas, pocos parámetros) para asegurar que cabe en el MCU.
2
Exportar a .tflite (cuantización INT8)
Convertir el modelo a TensorFlow Lite con cuantización INT8. Esto reduce el tamaño del modelo ~4x y acelera la inferencia en hardware de edge.
3
Abrir STM32Cube.AI en STM32CubeIDE
Iniciar STM32CubeIDE, abrir el proyecto target, y acceder al panel "X-CUBE-AI" desde la configuración del proyecto (.ioc).
4
Seleccionar STM32N6 como target
Configurar el microcontrolador objetivo (STM32N6) para que STM32Cube.AI genere código optimizado para su arquitectura específica.
5
Importar modelo .tflite
Cargar el archivo vibration_model.tflite generado. El wizard analizará automáticamente la arquitectura del modelo.
6
Analizar y validar el modelo
STM32Cube.AI estima el uso de RAM, Flash, y tiempo de inferencia. Se puede validar con datos de test para verificar que la precisión se mantiene tras la cuantización.
7
Generar código C optimizado
El tool genera automáticamente: network.c, network_data.c, y archivos de cabecera con el runtime de X-CUBE-AI.
8
Integrar en proyecto HAL
Incluir los archivos generados en el proyecto, implementar la lectura de sensores, llamar a aiRun() en el loop principal, y manejar las salidas.

Código Python: Entrenamiento del modelo de vibración

import tensorflow as tf
import numpy as np

# ============================================
# 1. Generar dataset sintético de vibración
# ============================================
np.random.seed(42)
# 500 muestras normales: baja vibración en 3 ejes
normal = np.random.normal(0, 0.5, (500, 3))
# 500 muestras anómalas: alta vibración en 3 ejes
anomaly = np.random.normal(2, 1.0, (500, 3))
# Combinar dataset
X = np.vstack([normal, anomaly])
y = np.array([0]*500 + [1]*500)  # 0=normal, 1=anomalía

# ============================================
# 2. Definir modelo tinyML (solo 3 capas)
# ============================================
model = tf.keras.Sequential([
    tf.keras.layers.Dense(16, activation='relu', input_shape=(3,)),
    tf.keras.layers.Dense(8, activation='relu'),
    tf.keras.layers.Dense(1, activation='sigmoid')
])

# ============================================
# 3. Compilar y entrenar
# ============================================
model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)
model.fit(X, y, epochs=20, batch_size=32, validation_split=0.2)

# ============================================
# 4. Cuantización INT8 para Edge AI
# ============================================
converter = tf.lite.TFLiteConverter.from_keras_model(model)
converter.optimizations = [tf.lite.Optimize.DEFAULT]
tflite_model = converter.convert()
open("vibration_model.tflite", "wb").write(tflite_model)
print("Modelo exportado: vibration_model.tflite")

Código de inferencia en C (STM32Cube.AI)

/* ============================================
 * Smart Factory Guardian - Inferencia Edge AI
 * STM32N6 + Ethos-U55 + Acelerómetro 3 ejes
 * ============================================ */

#include "ai_datatypes_defines.h"
#include "network.h"
#include "network_data.h"
#include "stm32n6xx_hal.h"

/* Buffers de entrada/salida alineados para la NPU */
__attribute__((section(".nn.tensor")))
__attribute__((aligned(4)))
ai_float in_data[AI_NETWORK_IN_1_SIZE];   // 3 valores flotantes (ax,ay,az)

__attribute__((section(".nn.tensor")))
__attribute__((aligned(4)))
ai_float out_data[AI_NETWORK_OUT_1_SIZE]; // 1 valor: probabilidad de anomalía

/* Handles del modelo X-CUBE-AI */
ai_handle network = AI_HANDLE_NULL;
ai_buffer ai_input[AI_NETWORK_IN_NUM] = {AI_BUFFER_INIT_NULL};
ai_buffer ai_output[AI_NETWORK_OUT_NUM] = {AI_BUFFER_INIT_NULL};

/* ============================================
 * 1. Inicializar el runtime X-CUBE-AI
 * ============================================ */
int AI_Init(void) {
    ai_error err;

    /* Crear instancia de la red neuronal */
    const ai_handle acts[] = { network_activations };
    err = ai_network_create(&network, AI_NETWORK_DATA_CONFIG);
    if (err.type != AI_ERROR_NONE) return -1;

    /* Asignar pesos y activaciones */
    const ai_network_params params = {
        AI_NETWORK_DATA_WEIGHTS(network_weights),
        AI_NETWORK_DATA_ACTIVATIONS(acts)
    };

    /* Inicializar la red con los parámetros */
    if (!ai_network_init(network, ¶ms)) return -2;

    /* Configurar buffers de entrada */
    ai_input[0] = ai_network_inputs_get(network, NULL)[0];
    ai_output[0] = ai_network_outputs_get(network, NULL)[0];

    return 0;
}

/* ============================================
 * 2. Leer acelerómetro (simulado por HAL)
 * ============================================ */
void Read_Accelerometer(float* ax, float* ay, float* az) {
    /* En hardware real: HAL_I2C_Mem_Read() del LIS3DH */
    /* Aquí leemos del sensor vía I2C/SPI */
    /* *ax = (int16_t)(buffer[0] | buffer[1]<<8) * 0.00098f; */
}

/* ============================================
 * 3. Normalizar datos de entrada
 * ============================================ */
void Normalize_Input(float ax, float ay, float az) {
    /* Normalización estándar: (x - mean) / std */
    float mean[3] = {1.28f, 1.08f, -0.52f};
    float std[3]  = {1.45f, 1.25f, 0.65f};

    in_data[0] = (ax - mean[0]) / std[0];
    in_data[1] = (ay - mean[1]) / std[1];
    in_data[2] = (az - mean[2]) / std[2];
}

/* ============================================
 * 4. Ejecutar inferencia con la NPU
 * ============================================ */
float AI_Run_Inference(void) {
    ai_i32 nbatch;
    ai_error err;

    /* Actualizar puntero de entrada con los datos normalizados */
    ai_input[0].data = AI_HANDLE_PTR(in_data);
    ai_output[0].data = AI_HANDLE_PTR(out_data);

    /* Ejecutar inferencia - delega automáticamente a la Ethos-U55 */
    nbatch = ai_network_run(network, &ai_input[0], &ai_output[0]);
    if (nbatch != 1) { return -1.0f; }

    return out_data[0];  // Probabilidad de anomalía [0.0 - 1.0]
}

/* ============================================
 * 5. Loop principal de la aplicación
 * ============================================ */
int main(void) {
    float ax, ay, az, confidence;

    HAL_Init();
    SystemClock_Config();
    AI_Init();           // Inicializar NPU y modelo
    Sensor_Init();       // Inicializar acelerómetro
    MQTT_Init();         // Conexión al broker

    while (1) {
        /* Leer sensor */
        Read_Accelerometer(&ax, &ay, &az);

        /* Preparar datos */
        Normalize_Input(ax, ay, az);

        /* Inferencia en la NPU (0.3ms) */
        confidence = AI_Run_Inference();

        /* Interpretar resultado */
        if (confidence > 0.5f) {
            /* ANOMALÍA detectada */
            Alarm_Activate();           // LED rojo + buzzer
            MQTT_Publish_Alert(ax, ay, az, confidence);
        } else {
            Alarm_Deactivate();         // Estado normal
        }

        HAL_Delay(100);  // Muestreo a 10 Hz
    }
}

/* Modelo tinyML: 3 entradas -> 16 -> 8 -> 1 salida */
const float weights_l1[3][16] = { /* ... */ };
const float bias_l1[16] = { /* ... */ };
const float weights_l2[16][8] = { /* ... */ };
const float bias_l2[8] = { /* ... */ };
const float weights_l3[8][1] = { /* ... */ };
const float bias_l3[1] = { /* ... */ };

float relu(float x) { return x > 0 ? x : 0; }
float sigmoid(float x) { return 1.0f / (1.0f + expf(-x)); }

float run_inference(float ax, float ay, float az) {
    float input[3] = {ax, ay, az};
    float hidden1[16], hidden2[8];

    /* Capa 1: 3 -> 16 (ReLU) */
    for (int i = 0; i < 16; i++) {
        hidden1[i] = bias_l1[i];
        for (int j = 0; j < 3; j++) hidden1[i] += input[j] * weights_l1[j][i];
        hidden1[i] = relu(hidden1[i]);
    }
    /* Capa 2: 16 -> 8 (ReLU) */
    for (int i = 0; i < 8; i++) {
        hidden2[i] = bias_l2[i];
        for (int j = 0; j < 16; j++) hidden2[i] += hidden1[j] * weights_l2[j][i];
        hidden2[i] = relu(hidden2[i]);
    }
    /* Capa 3: 8 -> 1 (Sigmoid) */
    float output = bias_l3[0];
    for (int j = 0; j < 8; j++) output += hidden2[j] * weights_l3[j][0];
    return sigmoid(output);
}

    Wire.beginTransmission(ACCEL_ADDR);
